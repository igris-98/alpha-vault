- AND
	- all true -> true
	- one false -> false
- OR
	- all false -> false
	- one true -> true
- XOR
	- same are false OR no of 1's are even
	- different are true OR no of 1's are odd
	-  0 ^ num = num
- NOT
	- flip bits
	- if negative 2's complement otherwise stop
- SHIFT
	- Right Shift (>>)
		x >> k = x / 2^k
	- Left Shift (<<)
		x << k = x * 2^k
